# Alumni-Networking-System
    ConnectsProject Name: Alumni Networking System

Project Description

The Alumni Networking System is a database management project designed to connect alumni and students through a centralized online platform. It allows users to register, create and manage profiles, search alumni information, and communicate through a messaging system. The project helps strengthen professional and academic connections among alumni and current students.

Features

1. Registration & Login
    * User account creation and secure login.
    * Authentication using email and password.
2. Profile Management
    * Users can create, update, and manage their personal and professional information.
    * Profile details include name, graduation year, department, and contact information.
3. Alumni Directory
    * Search and view alumni profiles.
    * Filter alumni by department, batch, or graduation year.
4. Messaging
    * Send and receive messages between registered users.
    * Enables networking and communication among alumni.
